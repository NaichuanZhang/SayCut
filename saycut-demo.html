AI Pipeline

6 models, one voice command

1 higgs-audio-v3.5 Voice → text + tool calls (BosonAI)
2 kimi-k2-5 Script generation with scene structure (EigenAI)
3 eigen-image Text-to-image for each scene (EigenAI)
4 qwen-image-edit Inpainting for voice-directed edits (EigenAI)
5 wan2p2-i2v-14b Image-to-video animation, async polling (EigenAI)
6 higgs2p5 Multi-voice TTS with WAV concatenation (EigenAI)
Tool Calling

Voice agent orchestrates everything

1
Tool Injection Tools defined as JSON in <tools> block within system prompt
2
Model Emits <tool_call> Parsed via regex, normalized from array/object/nested formats
3
Local Execution Async tool executor runs generate_script, generate_image, etc.
4
<tool_response> Loop Result fed back as user message — up to 6 chained rounds
5
Auto-Nudge If model narrates instead of acting, system re-prompts: "call the next tool"
